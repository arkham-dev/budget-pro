# 💰 Budget Pro

Application de budget personnel — PWA mobile-first

![Status](https://img.shields.io/badge/status-live-brightgreen)
![PWA](https://img.shields.io/badge/PWA-ready-blue)
![License](https://img.shields.io/badge/license-proprietary-red)

## Fonctionnalités

- Suivi du solde en temps réel
- Transactions récurrentes (hebdo, 2 sem., mensuel, annuel)
- Calendrier budgétaire mensuel
- Détection automatique des risques financiers
- Mode hors-ligne (PWA installable)
- 100% local — aucune donnée envoyée

## Technologies

- HTML5 / CSS3 / JavaScript vanilla
- localStorage pour la persistance
- Service Worker pour le mode hors-ligne
- PWA installable sur iOS et Android

## Déploiement

Hébergé sur Netlify — déploiement automatique depuis GitHub.
